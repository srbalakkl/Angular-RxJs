Distinct Until Changed Operator.

  1. distinctUntilChanged compare the current value with the previous value to produce an output.
  2. It works just like === operator
  3. Attention:It can compare only primitive values, comparison like object == object will be ignored.
  4. Use DistinctUntilKeyChanged Operator to compare an object.
      5. Or use the distinctUntilChanged like this for object
  5. distinctUntilChanged((prev, curr) => prev.name === curr.name)
